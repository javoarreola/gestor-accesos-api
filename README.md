# Gestor de Accesos API

API REST desarrollada con FastAPI para administrar el registro de ingresos y salidas de visitantes dentro de una empresa.

## Integrantes

- Javier Arreola Fonseca
- Juan Ricardo Felix

## Tecnologías utilizadas

- Python
- FastAPI
- SQL Server
- SQLAlchemy
- PyODBC
- Git
- GitHub

## Objetivo del proyecto

Diseñar y desarrollar una API REST que permita registrar, consultar y gestionar accesos de visitantes, proveedores, contratistas y personal externo dentro de una organización.
